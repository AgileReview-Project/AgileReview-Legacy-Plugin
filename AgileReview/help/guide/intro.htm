AgileReview overview

The Agilereview (AR) project provides a tool plug-in to review documents (which can be opende in the Eclipse IDE) in fast and easy way. It adds a Review project and AR perspective to the Eclipse Workbench as well as a number of views, annotations, wizards and refactoring tools.

Try the quickstart to get a first impression. Discover more in the Tips and Tricks section.
