Quickstart

The following is a quick start guide to the AgileReview project, an eclipse plugin for code-reviewing. It is meant to walk you through the basic steps to use this plugin seriously. This means to do code-review in a most efficient way.

Short Introduction

This plugin is used to do code reviewing in your development environment (Eclipse). So it is possibel to switch between the development perspective and the code-reviewing perspective as quick as it is possible. There is no more effort in setting up an own review environment in another software (eg Excel). This code-reviewing is working with two different levels:

  • review level:
    On this level it is possibe to create, change and delete reviews. A review is a (big) section of the code or project in which a special reviewer surveys the developed code and gives comments to this.
  • comment level:
    On this level it is possibel to create, change and delete comments. These comments are the comments the reviewer wants to make regarding the developed code. Therefore it is possible to have a lot of comments under only one review

Using Agilereview

We try to provide a cheatsheet for all of the now mentioned actions in the help section.

Create new Review

  1. Go into the ReviewExplorer
  2. Go on the add-new-review icon
    --> NewReview wizard starts
  3. Insert data for the review (Especially the ReviewName can not be empty)
  4. Confirm this data with Finish

Now the Review with its date is created and shown in the ReviewExploere as acive review. And also the Detail view shows the inserted detail data of this review.

Change Review

  1. Go into the ReviewExplorer
  2. Select the review you want to change
    --> the detail view opens the selected review
  3. Go into the detail view
  4. Change the data

Now, when the detail view looses the focus the next time all the changes you made will be saved.

Delete Review

  1. Go into the ReviewExplorer
  2. Select the review(s) you want to delete
  3. Press DELor the delete-icon to delete thee selected review(s)

Create new Comment

  1. Activate a review in the ReviewExplorer
  2. Select the code line(s)/section where the comment should be placed
  3. Go on the add-new-comment icon
    --> Comment detail opens
  4. Insert comment data

Now, when the detail view looses the focus the next time all the changes you made will be saved. The inserted comment is shown in the CommentSummary view with most of the data.

Change Comment

  1. Go into the CommentSummary view
  2. Select the comment you want to change
    --> the detail view opens the selected comment
  3. Go into the detail view
  4. Change the data

Now, when the detail view looses the focus the next time all the changes you made will be saved.

Delete Comment

  1. Go into the CommentSummary view
  2. Select the comment(s) you want to delete
  3. Press DELor the delete-icon to delete the selected review(s)
